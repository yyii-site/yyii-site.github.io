1.Advanced users - Build and install

  2. 若使用自己修改的TheengsDecoder，注意在setup.py文件中修改依赖包的版本。
  3. setup.py修改version版本号，例如version="1.1.0"
  4. __main__.py调试项目，提示import失败，需将from . import main改为from TheengsGateway import main
