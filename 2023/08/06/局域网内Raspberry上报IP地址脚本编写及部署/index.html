局域网内Raspberry上报IP地址脚本编写及部署

yi
2023-08-06
RaspberryPi

IP UDP Python Shell

1.安装python包

2.将脚本复制到指定的目录

/usr/local/bin/send_ipaddr_broadcast.py

3.新建系统服务

  • 将send_ipaddr_broadcast.py复制到/etc/systemd/system/

  • 开启服务

  • 开机自启

4.测试

  • 打开网络调试助手

  • 选择UDP模式

  • 选择合适的本地主机地址（非127.0.0.1）

  • 输入本地主机端口7893

  • 点击开启按钮

  • 接收设置选择：ASCII模式
